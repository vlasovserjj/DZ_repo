Git

  • git clone ссылка с GitHub - позволяет клонировать удаленный репозиторий себе на компьютер
  • git -v - эта команда выведет версию Git, если он правильно установлен.
  • git init - команда инициализирует репозиторий
  • dir /ah - проверка наличия скрытых файлов в репозитории
  • git status - команда для просмотра изменений в репозитории
  • git add название файла - команда позволяет сделать файл отслеживаемым
  • git commit -m "first commit" - команда сохраняет изменения в репозитории
  • git push - команда вносит изменения на удаленный репозиторий
